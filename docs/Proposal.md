## 중·고등학생 학습 챌린지 소셜 플랫폼 - Study Mate

### 1. 개요

혼자 공부하기 어려운 학생들을 위해 하루 학습 인증과 친구 간 경쟁/응원 시스템을 제공하는 학습 습관 챌린지형 플랫폼입니다.

학생은 공부 목표를 챌린지로 설정하고 매일 학습 인증을 올리며 친구들과 함께 꾸준함과 성취감을 시각적으로 확인할 수 있습니다.

### 2. 문제 상황

- 혼자 공부하는 학생들은 계획은 세우지만 꾸준히 실행하지 못
- SNS는 교류 중심이라 학습에 집중이 어려움, 오히려 방해 요인으로 작용함
- 기존 습관앱은 타인과의 연결성/응원 요소가 약함
- 작은 성취가 축적/시각화되지 않아 동기 유지가 어려움

### 3. 기획 배경

- 교육 시장과 습관 추적 [앱 시장 규모](https://www.businessresearchinsights.com/ko/market-reports/habit-tracking-apps-market-109438)는 꾸준히 성장 중
- 중·고등학생은 스마트폰 사용률이 높고 SNS형 UI에 익숙함

- 이를 기반으로 SNS 동기부여 요소(피드와 응원)와 습관앱의 구조(목표와 주기)를 결합해 학습 지속력을 높이는 서비스를 기획

### 4. 주요 기능

#### 4.1 챌린지 생성 및 참여

- 사용자가 직접 챌린지를 만들거나 친구의 챌린지에 참여
  - ex) “매일 영어 단어 30개 외우기”, “수학 문제집 하루 2장”, “국어 비문학 지문 1개 읽기”
- 기간, 목표(주당 횟수) 등

#### 4.2 학습 인증

- 하루 한 번 학습 인증 업로드
- 입력:
  - 제목, 학습 목표, 학습 내용 요약, 오늘 배운 점, 자료 등

#### 4.3 소셜 기능 및 시각화

- 소셜 기능 (팔로우, 좋아요): 다른 사용자를 팔로우하여 그들의 챌린지 현황을 구독하고, '응원해요(좋아요)'를 통해 상호작용
  - 댓글 기능은 추후 확장
- 챌린지 달성률 시각화: 차트 형태(charts.js)로 시각화

#### 4.5 치팅 방지 및 신고

- 치팅 방지
  - 인증 성공 판정은 서버 수신 시각(00:00~23:59)
  - 챌린지별 1일 1회 업로드 제한
- 신고 기능
  - 신고 버튼 – 사유 선택(광고/음란/혐오/폭력/기타)

### 5. 사용 시나리오

1. A가 “수학 문제집 하루 2장 풀기” 챌린지를 개설
2. 친구 B가 챌린지에 참여
3. 매일 공부 후 간단한 요약 업로드
4. 친구들이 피드에서 응원
5. A와 B는 “연속 학습일수”와 “주간 성취 그래프”를 확인
6. 좋아요/응원 등 누를 수 있음
   A. 댓글은 차후 추가
