## IT 학습 챌린지 소셜 플랫폼 - Study Mate

### 1. 개요

개발자 지망생과 초보 개발자/학습자가 직접 IT 학습 챌린지를 만들고 참여하며, 학습 정리 노트나 링크를 통해 학습 인증을 통해 꾸준함을 유지하도록 돕는 소셜 서비스

### 2. 문제 상황

- 대부분의 사람들은 혼자서 학습을 지속하고 복습하며 정리하기 어려움
- 기존 SNS는 교류 중심이고 습관앱은 타인과의 연결성이 약함
- 작은 성취가 축적/시각화되지 않아 동기 유지가 어렵다.

### 3. 기획 배경

- 자기계발 관심이 높아졌고, 그 중 학습 수요가 두드러짐.
- 최근 AI의 발달으로 바이브코딩에 대한 수요가 높아짐
- SNS 동기부여 요소(피드와 응원)와 습관앱의 구조(목표와 주기)를 결합해 학습 지속력을 높이는 서비스를 기획

### 4. 주요 기능

#### 4.1 챌린지 생성/인증

- 챌린지 생성 및 참여: '매일 알고리즘 1문제', 'JS 30분 강의' 등 자신만의 챌린지를 만들거나 다른 사람의 챌린지에 참여
- 인증(IT 특화):
  - 제목, 학습 목표, 학습 내용 요약, 오늘 배운 점, 자료 등
  - GitHub나 Blog 링크

#### 4.2 소셜 기능 및 시각화

- 소셜 기능 (팔로우, 좋아요): 다른 사용자를 팔로우하여 그들의 챌린지 현황을 구독하고, '응원해요(좋아요)'를 통해 상호작용
  - 댓글 기능은 추후 확장
- 챌린지 달성률 시각화: 차트 형태(charts.js)로 시각화

#### 4.3 치팅 방지 및 신고

- 치팅 방지
  - 인증 성공 판정은 서버 수신 시각(00:00~23:59)
  - 챌린지별 1일 1회 업로드 제한
- 신고 기능
  - 신고 버튼 – 사유 선택(광고/음란/혐오/폭력/기타)

### 5. 사용 시나리오

1. A가 '매일 알고리즘 1문제' 챌린지 생성(기간 정함)
2. A를 팔로우했던 B가 해당 챌린지를 보고 참여
3. 챌린지 수행 후 목차에 따라 학습 내용 정리해서 업로드
4. 인스타 피드 형태로 공유됨
5. 좋아요/응원 등 누를 수 있음
   A. 댓글은 차후 추가
