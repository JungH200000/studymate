# 추가적인 기능 요소 후보

## 동기부여 요소

### 달성률 랭킹

-
- 최근 30일을 집계한 달성률로 자신의 팔로워와 팔로잉의 달성률 랭킹을 매김
- 실시간으로 집계하여 등수 변동 사항을 확인할 수 있음(등수 옆에 +1 또는 -2 등)

- 기대 효과: 팔로우 관계 기반 경쟁으로 사용자의 동기를 부여하여 챌린지 지속 참여에 기여

## AI

### AI 욕설·비방 검증

- SNS 안전 확보를 위해 AI로 사용자가 올린 인증글에 욕설·비방이 담겨있지 않은지 판단

- 개선 방향: 고도화하여 AI 인증 검증으로 발전하여 챌린지에 맞는 인증글을 작성했는지, 잘못된 내용을 기입하지 않았는지 등 피드백 절차 마련

### AI Report

- 버튼을 눌러 AI로 사용자가 올린 챌린지 및 인증글을 바탕으로 학습 보고서 제공(월간/연간 등)
