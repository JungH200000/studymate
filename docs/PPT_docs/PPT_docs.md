그런데 지금 학생들이 가장 많이 사용하는 도구인 SNS는 학습 목표/진도/동기 설계 기능이 부족하고, 습관앱은 동료 기반 상호작용이 약해 꾸준함을 만들기 어렵습니다. 결국 핵심은 혼자 공부하는 구조에 있다고 보았습니다.
