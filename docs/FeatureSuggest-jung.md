# 추가적인 기능 요소 후보

## 동기부여 요소

### 1. 달성률 랭킹

- 저번 주까지 집계한 전체 달성률로 자신의 팔로워와 팔로잉의 달성률 랭킹을 매김 -> 경쟁으로 동기 부여하여 참여 의지를 높임

### 2. 재촉하기

- 챌린지에 참여한 유저들 중에 인증글을 올리지 않은 유저에게 '재촉하기' 같은 버튼 추가해서 누르면 해당 유저에게 알림 또는 앱 안에서는 모달창이 출력되도록 설정

## AI

### 1. AI Coach

- AI로 사용자가 올린 인증글 JSON에서 goals와 summary 분석하여 다음 학습 팁을 생성

### 2. AI Report

- AI로 사용자가 올린 챌린지 및 인증글을 바탕으로 학습 보고서 제공(주간/월간/연간 등)

### 3. 추천 챌린지 목록

1. 사용자의 최근 n개의 인증글을 AI(Gemini API 또는 GPT API)가 분석하여 관련 tag 추출(현우진, 이미지 같은 강사 이름이나 수I, 물리 같은 과목 이름)

2. tag를 바탕으로 챌린지를 sort하여 home화면에 추천 챌린지가 출력 + tag table에 AI가 분석한 해당 사용자의 tag 저장(사용자가 로그인할 때마다 또는 특정 주기마다 갱신)
   - 챌린지의 content를 jsonb로 변경 후 tag 추가
