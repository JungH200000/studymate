## 중·고등학생 학습 챌린지 소셜 플랫폼 - Study Mate 기획서

### 1. 개요

혼자 공부하기 어려운 학생들을 위해 하루 학습 인증과 친구 간 경쟁·응원 시스템을 제공하는 학습 습관 챌린지형 플랫폼입니다.

학생은 공부 목표를 챌린지로 설정하고 매일 학습 인증을 올리며 친구들과 함께 꾸준함과 성취감을 시각적으로 확인할 수 있습니다.

### 2. 기획 의도 및 배경

1. 자기주도 학습 지속 한계

- 많은 학생이 목표는 알차게 세우지만 꾸준히 실천하는 데 어려움을 겪습니다.
- [자료1] 서울대 교육학과 신종호 교수는 자기주도 학습이 흔히 실패하는 이유로 "공부를 하는 이유에 대한 목적의식의 부재", "흥미 결여", "혼자 공부하기의 난이도"를 지적합니다.
- 따라서 학생들에게는 학습을 지속하게 만들 공부 이유와 필요성이 필요합니다.

2. 기존 학습 도구의 한계

- 타이머 중심 기능
  - 기존의 학습앱(열품타, Flipd)은 공부 시간 측정과 집계에만 집중되어 실제 학습 진도와 성취도를 보장하지 못합니다.
  - 이는 시간만 채우는 보여주기식 경쟁으로 이어질 수 있습니다.
- 과제 단위 관리 부재
  - 학생의 공부는 '단어 암기', '문제집 풀이' 같은 구체적인 단위로 이루어집니다.
  - 그러나 기존 앱은 과목별·단계별 목표 설정 기능이 부족하며 수행한 과제를 단순 총 공부 시간으로 합산하여 기록하는 방식입니다.
- 동기부여 부족 및 상호작용 부족
  - 단순 시간 기록과 랭킹외에는 구체적인 학습 내용에 대한 피드백이나 인증 수단이 부족합니다.
  - 서로의 학습 내용을 확인하고 서로 응원하는 상호작용 없어 함께 공부한다는 느낌을 주기 어렵습니다.

3. 해결 방안

그래서 저희는 단순 시간을 측정하는 것이 아닌 구체적인 목표를 챌린지로 설정하고 그 결과를 인증하고 공유하는 행동 중심 플랫폼 Study Mate를 기획했습니다.

- 공부 시간 경쟁이 아닌 오늘 수행한 학습 목표를 인증함으로 실질적인 성취감을 부여하고,
- 친구들과 서로 인증글을 확인하고 응원하며 함께 성장하는 환경을 만들고,
- 달성률 그래프와 인증 기록으로 자신의 누적 성과를 시각적으로 확인하고 지속적인 동기를 얻도록 설계했습니다.

### 3. 주요 기능

#### 3.1 챌린지 생성 및 참여

- 사용자는 목표(제목, 시작일, 주간 빈도)를 입력해 챌린지를 쉽게 개설하고 상세 설명이나 종료일, 태그는 선택적으로 설정
- 홈에서 최신 챌린지를 탐색하며 참여자 수, 좋아요 수, 친구 목록을 확인하고 곧바로 참여하거나 취소할 수 있음
- 개인의 참여 여부와 챌린지의 활동 지표를 실시간으로 갱신해 버튼 상태와 참여자 명단을 직관적으로 보여줌
  - 챌린지의 활동 지표 : 좋아요·참여자·인증글 수
- 챌린지를 누르면 상세 정보와 인증글 목록이 탭으로 분리된 화면이 출력되고, 기간·빈도·성과 지표를 먼저 확인하거나 인증글 목록으로 바로 이동할 수 있음

#### 3.2 학습 인증

- 챌린지 참가자는 하루 한 번, JSON 기반 인증글을 작성해 목표·요약·자료·학습 시간 등 학습 기록을 자유롭게 남김
- 서버가 참여 여부와 날짜 중복을 검증해 미참가자나 하루 1회 제한을 넘는 업로드를 차단, 공정한 인증 흐름을 보장
- 업로드 직후 챌린지 전체 인증 수와 나의 누적·주간 실적, 주간 목표 대비 진행률을 계산해 달성 현황에 반영
- 인증글 목록은 최신순·페이지네이션으로 제공되며 각 글의 응원 수, 응원자 목록, 내 응원 여부를 함께 보여 학습 공유를 도움

#### 3.3 소셜 기능 및 시각화

- 친구를 팔로우해 서로의 챌린지 활동을 구독하고 팔로워·팔로잉 목록에서 학습 네트워크를 관리
- 챌린지에는 좋아요, 인증글에는 응원하기를 제공해 간단한 피드백으로 참여 의지를 높임
- 사용자 페이지에서는 참여·등록 챌린지별 주간 달성률을 막대 그래프로 제공하고 달성률을 탭하면 전체·최근 30일 달성률 그래프와 일별 누적 인증 횟수를 보여주는 GitHub 잔디형 캘린더 페이지로 이동해 누적 학습 성과와 현재 진행 상황을 한눈에 확인
- 최근 30일 달성률을 기준으로 팔로워·팔로잉 간 랭킹을 실시간 갱신(+1/−1 변동 표기)해 경쟁 동기를 강화하여 챌린지 참여 의지를 유지

#### 3.4 치팅 방지·신고

- 치팅 방지
  - 서버 수신 시각 기준(00:00~23:59)으로 인증 성공 판정, 챌린지별 1일 1회 업로드 제한
- 신고 기능
  - 신고 버튼 – 광고·음란·혐오·폭력·기타 등에 관련하여 신고하고 신고 사유 작성

#### 3.5 AI 추천 챌린지

- 인증글 업로드 시 백그라운드에서 Gemini API가 인증글을 분석해 과목·강사·개념 같은 태그를 최대 5개 추출
- 태그는 사용자별로 점수처럼 누적됨
- 홈 추천 목록은 사용자의 태그와 챌린지의 태그가 많이 겹칠수록 위에 노출

#### 3.6 AI 욕설·비방 검증

- 인증글 업로드 시 SNS 안전 확보를 위해 Gemini API로 욕설·비방 등 유해 표현을 사전 차단

### 4. 사용 시나리오

1. A가 “수학 문제집 하루 2장 풀기” 챌린지를 개설
2. 친구 B가 챌린지에 참여
3. 매일 공부 후 간단한 요약 업로드
4. 친구들이 인증글 목록에서 응원
5. A와 B는 “주간·전체 달성률”을 그래프로 확인
6. 좋아요·응원 등 누를 수 있음
   - 댓글은 차후 추가 예정
7. A 사용자가 홈 화면에서 추천 챌린지 탭을 열어, 이 때까지 작성한 인증글에 기반해 추출한 개인 태그와 챌린지 태그의 매칭 결과로 정렬된 챌린지 목록을 확인
