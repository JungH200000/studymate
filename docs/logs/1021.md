# Register

- request

```json
{
  "email": "chungha@naver.com",
  "password": "1234",
  "username": "chungah"
}
```

- response

```json
{
  "message": "✅ Register Success",
  "user": {
    "user_id": "5a73b52c-687d-48d8-94cf-708ceeb6522f",
    "email": "chungha@naver.com",
    "username": "chungah",
    "created_at": "2025-10-21T07:26:40.572Z"
  }
}
```

[O] `email`, `username` 중복되면 오류 출력
[O] 유효하지 않은 email -> 오류 출력
