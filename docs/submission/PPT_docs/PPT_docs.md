### 예상 슬라이드 내용

- 자기주도 학습 지속 한계와 고립된 환경

- 1. 챌린지 탐색 및 참여

  - "목표를 설정하여 챌린지에 참여"
  - 목표 설정 및 개설 : 사용자가 학습 목표와 기간, 빈도(주간/일간), 태그를 설정해 직접 챌린지를 생성
  - 참여·취소 토글 : 버튼 하나로 간편하게 챌린지에 참여하거나 취소
  - 활동 지표 확인 : 참여자 수, 좋아요 수 등 활동 지표를 실시간으로 확인하여 동기 부여

- 2. 학습 인증과 AI 검증·분석

  - "학습 내용을 기록하고 AI가 검증 및 분석"
  - 학습 내용 기록 : 챌린지별 하루 1회, JSON 기반으로 학습 목표·요약·시간을 기록하여 인증
  - AI 유해 표현 필터링 : Gemini API가 내용을 검증하여 욕설·비방 등 유해 표현을 서버에서 자동 차단
  - 인증글 목록 : 챌린지별 인증글을 확인하고 친구들과 응원을 주고 받으며 지속적인 동기 부여
  - AI 추천 챌린지 : Gemini API가 인증글을 분석해 학습 주제 태그를 자동 추출하고, 이를 추천 데이터로 활용하여 사용자별 맞춤 챌린지를 추천

- 3. 성과 확인 및 경쟁

  - "달성률을 시각화하고 랭킹으로 경쟁"
  - GitHub 잔디형 캘린더 : 일별 인증 횟수를 잔디 형태로 시각화하여 학습 패턴과 빈도를 파악
  - 주간·누적 달성률 시각화 : 인증 횟수 기반으로 달성률을 계산해 그래프로 누적 학습 성과를 시각화
  - 팔로우 기반 실시간 랭킹 : 친구들과 팔로워·팔로우 관계를 기반으로 실시간 랭킹 경쟁

## 12 슬라이드 : 시스템 개발 내용-서비스별 설명-AI 추천 챌린지

<img src="./image/12-1. 데이터 확보.png" width=250px>
<img src="./image/12-2. 관심사 분석.png" width=250px>
<img src="./image/12-3. 추천 챌린지 목록.png" width=250px>

1. 데이터 확보
   - AI 태그 추출
     - 사용자가 인증글을 업로드하면 Gemini API가 백그라운드(Queue)에서 텍스트를 분석
     - Output: 학습 주제 태그 추출 (예: #영어 #한국사 #알고리즘 등)
2. 관심사 분석
   - 가중치 누적
     - 추출된 태그는 `user_tags` 테이블에 저장되며 빈도에 따라 가중치(weight)가 누적
     - “이 사용자가 무엇을 자주 공부하는가?”를 시스템이 정량적으로 학습
3. 추천 챌린지 목록
   - 태그 매칭
     - 누적된 관심사 데이터와 챌린지 태그를 매칭하여 개인에게 최적화된 챌린지를 상단에 노출
     - Result: 인증 활동이 챌린지 탐색을 돕는 구조가 완성됨

## 15 슬라이드 : 시연 영상 및 기대효과-프로젝트 기대효과

### 작성 사항

- 프로젝트 수행을 통해 기대되는 예상 기대효과

### 예상 슬라이드 내용

- 결과 중심 습관 고정
  - 1일 1회 인증과 정형 기록으로 단순히 책상에 앉아있는 것이 아니라 매일 구체적인 학습 결과물을 만들어내는 목표 완료 학습 습관으로 전환
- 안전한 연대·지속 동기
  - Gemini API로 유해 표현을 차단하여 안전한 소통을 보장하고, 응원과 팔로워 기반 랭킹 시스템이 페이스메이커 역할을 하여 고립감을 줄이고 지속적인 동기 부여
- 데이터 기반 자기 주도력 강화
  - 잔디형 캘린더와 주간·전체·최근 30일 달성률 그래프로 학습 패턴을 파악하고, 인증 태그 기반 AI 추천 챌린지로 내 학습 태그와 맞는 과제를 계속 이어가며 자기 주도력을 강화

## 16 슬라이드 : 시연 영상 및 기대효과-향후 계획 및 보완사항

### 작성 사항

- 프로젝트 향후 계획 및 보완사항

### 예상 슬라이드 내용

1. 푸시 알림·학습 타이머 연동
   - 미인증 참여자 대상 '재촉하기' 알림 도입으로 챌린지 동기 유지
   - 학습 타이머 연동을 통해 인증 기록의 정확성 향상
2. AI 인증 검증 고도화
   - 욕설 차단을 넘어 챌린지 적합성·허위 여부·학습 피드백까지 자동 검증하는 절차 마련
3. 오프라인 커뮤니티 연계 확장
   - 학교, 학원뿐만 아니라 독서실, 스터디 카페, 동호회 등 다양한 오프라인 그룹과 연계된 전용 챌린지 및 랭킹 도입
4. 학습 리포트 생성
   - 누적 인증 데이터를 기반으로 주간/월간 리포트를 자동 발급하고, 학습 방향 설계 자료로 활용
