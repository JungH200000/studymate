# Feedback01

## 3 슬라이드 : 프로젝트 개요-추진배경

- "학생들이 왜 혼자 공부하기 어려울까?"에 대한 대답이 충분하지 않은 듯함
  - 자기주도 학습의 난관과 기존 도구의 한계는 어느정도 설득력 있는 듯함
  - 그러나 디지털 환경의 역설과 주의력 지우너 필요성 증가는 해당 프로젝트로 해결할 수 없는 문제이기 때문에 언급 불필요
  - 국내외 경쟁앱을 찾아보면 유명한 앱들이 존재하지만 대부분 학습을 얼마나 했는지 타이머에 포커싱 되어 있음 -> 기존앱의 한계점을 얘기하고 단점을 보완하는 방식으로
    - ex) 시간 측정 중심으로 과제 단위 진도 관리 / 인증 / 랭킹 등 종합 구조를 제공하지 못함
    - Athenify, Flipd, studytogether 등등
    - 열품타, 플립

## 9 슬라이드 : 시스템 개발 내용-사용자 인터페이스(UI)

- 단순히 기능만 먼저 설명하지 말고 구체적으로 우리 솔루션이 가지고 있는 핵심 기능을 한 페이지로 정리하고 그것을 뒤에서 설명하는 방식으로 만들면 좋을 듯
- 해당 슬라이드에 5개 기능이 있는데 이게 발표자 있는 PPT가 아니기 때문에 최대한 읽어봤을 때 구체적으로 적어야 함
- 각각의 메뉴들이 어떤 역할을 하는지 적어주는게 좋을 것 같음
- 각 메뉴에 대해서 뒷슬라이드에서 스크린샷과 함께 세부적으로 설명할 예정이니 총괄로 한 눈에 보일 수 있도록 해주면 좋을 듯

## 15 슬라이드 : 시연 영상 및 기대효과-프로젝트 기대효과

- 좀 더 구체적으로 작성할 필요가 있음
- 단순히 뭐뭐를 해서 동기부여가 가능하다에서 좀 더 구체적으로
- 예를들어 "지속 참여 촉진이 아니라 어떻기 때문에 지속적으로 학생들이 참여 가능하다"식으로 구체적으로 작성하면 좋을 듯
