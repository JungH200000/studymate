# 중·고등학생 학습 챌린지 소셜 플랫폼 - Study Mate

혼자 공부하기 어려운 학생들을 위해 하루 학습 인증과 친구 간 경쟁/응원 시스템을 제공하는 학습 습관 챌린지형 플랫폼

---

## 개발 의도

중, 고등학생들은 계획은 세워도 혼자 꾸준히 실천하고 복습하기 어렵습니다. 일반 SNS는 교류와 오락 중심이라 학습 집중에 방해가 되고, 기존 습관앱은 타인과의 연결성과 응원 요소가 약합니다. **Study Mate**는 SNS의 동기 부여 요소(피드와 응원)와 습관앱의 구조(목표와 주기)를 결합해 매일의 학습 인증을 친구와 함께 축적하고 시각화함으로써 지속성을 유지하기 위한 서비스입니다.

---

## 역할 분담

- 박정현 : UI/UX 설계, Server 설계 및 구현, DB 설계 및 구현, PPT 제작
- 김지훈 : UI/UX 설계, Frontend 구현, UI/UX 구현
- 한진환 : UI/UX 설계, Frontend 구현

---

## 기술 스택

- **Frontend**: React, Morpheus(유라클)
- **Sever**: Node.js(Express), JWT
- **DB**: PostgreSQL(Supabase)
- **Dev Tool**: VS Code, Android Studio, Postman
